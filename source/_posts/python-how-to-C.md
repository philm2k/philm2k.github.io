# C

## Closure(클로저)

{% blockquote 루시아누 하말류, 전문가를 위한 파이썬 %}
클로저는 함수 본체에서 정의하지 않고 참조하는 비전역(nonglobal) 변수를 포함한 확장 범위를 가진 함수이며 익명 함수일 필요는 없다. 함수 본체 외부에 정의된 비전역 변수에 접근할 수 있다는 것이 중요하다.
{% endblockquote %}
